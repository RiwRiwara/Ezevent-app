# Ezevet Mobile application
## Requirement
- Node version : v18.12.1
- NPM version : 9.7.1
  
## Install project dev
1. Install by requirements.
2. clone project
   ```
   git clone https://github.com/RiwRiwara/Ezevent-app
   ```
3. install dependency
   ```
   npm install
   ```
   ```
   npm update
   ```
4. run application
   ```
   npx expo start
    ```
